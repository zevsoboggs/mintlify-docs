## Базовый URL

```
https://sand.tranzor.io
```

Все запросы к API выполняются по HTTPS. HTTP-запросы не поддерживаются.

## Аутентификация

API поддерживает два способа аутентификации:

- **Bearer Token** — заголовок `Authorization: Bearer trz_...`
- **HMAC-подпись** — заголовки `X-API-Key`, `X-Timestamp`, `X-Signature`

Подробнее в разделе [Аутентификация](/authentication).

## Формат ответов

Все ответы возвращаются в формате JSON. Успешный ответ:

```json
{
  "success": true,
  "data": { ... }
}
```

Ответ с ошибкой:

```json
{
  "success": false,
  "error": "Описание ошибки"
}
```

## Коды ошибок

| Код | Описание |
|-----|----------|
| `400` | Невалидный запрос — проверьте параметры |
| `401` | Не авторизован — проверьте API ключ или подпись |
| `404` | Ресурс не найден |
| `500` | Внутренняя ошибка сервера |

## Статусы инвойсов

| Статус | Описание |
|--------|----------|
| `PENDING` | Ожидает оплаты |
| `PAID` | Оплачен |
| `EXPIRED` | Истёк срок оплаты |
| `CANCELLED` | Отменён |
